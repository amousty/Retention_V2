Nom 		: login.php

Rôle 		: Compare le login et le mdp provenant du formulaire avec la base de donnée 
		  et envoit un code en conséquence (si ça s'est bien passé ou non).

Entrée(s) 	: - Le login
		  - Le mot de passe

Sortie(s) 	: - Le code permettant de savoir si le login s'est bien passé.
		  - Ok si authentification correcte.
		  - ERR si le login et/ou le mdp sont incorrects.
